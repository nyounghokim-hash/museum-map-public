## Login Notes

Map exploration, search, museum detail pages, MM Story, and artwork pages can be reviewed without login. Saved museums, collections, reviews, and trip plans require Google OAuth authentication.

## Public Repository Scope

This public repository is a sanitized copy of the private production repository. It includes evaluation-safe source code and documentation, but excludes environment variables, production secrets, raw logs, scratch scripts, and deployment-only operational notes.
