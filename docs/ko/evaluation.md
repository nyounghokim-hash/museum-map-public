# Museum Map 평가 요약

## 프로젝트 개요

Museum Map은 전 세계 3,840+개 박물관과 미술관을 지도에서 탐색하고, AI 추천을 받고, 여행 동선을 계획할 수 있는 다국어 플랫폼입니다.

## 라이브 데모

https://museummap.app

## 현재 현황

| 항목 | 내용 |
| --- | --- |
| 공개 박물관/미술관 | 3,840+ |
| 박물관 사진 캐시 | 공개 박물관 100% 커버리지 |
| 작품 | 900+ |
| MM Story | 120+ |
| 지원 언어 | 13개 |
| 앱 버전 | 1.6.1 |

## 평가 방법

1. https://museummap.app 에 접속합니다.
2. 메인 지도에서 도시, 국가, 박물관 이름으로 검색합니다.
3. 박물관 상세 패널에서 사진, 관람 정보, 리뷰, 관련 스토리, 길찾기를 확인합니다.
4. AI 추천에서 자연어 검색을 시도합니다.
5. MM Story와 작품 상세 페이지를 확인합니다.
6. 언어 전환을 확인합니다.

## 로그인 안내

지도 탐색, 검색, 박물관 상세, MM Story, 작품 페이지는 로그인 없이 확인할 수 있습니다. 저장, 컬렉션, 리뷰, 여행 계획 기능은 Google OAuth 로그인이 필요합니다.

## 공개 repo 범위

이 공개 repo는 private production repository에서 평가 가능한 코드와 문서만 분리한 sanitized copy입니다. 환경변수, production secret, raw logs, scratch scripts, deployment-only notes는 포함하지 않았습니다.
